# aladinTVLogos

This repository contains 96px WebP television channel logos optimized for IPTV applications.

## Project Structure

- All logos are formatted as `.webp` files.
- Dimensions: 96x96 pixels.
- Filenames are standardized for easy dynamic mapping.

## Keep Workspace Active

To prevent GitHub Codespaces from automatically deleting this environment due to inactivity, a Python script is included to refresh file retention status.

Run the keep-alive script:
```bash
python keep_active.py
